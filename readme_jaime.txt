Para jugar a pongoline.

	1. Iniciar el servidor
	2. Iniciar un cliente y ponerle nombre.
	3. Cambiar el puerto del socket del cliente(da igual el numero) y darle a iniciar.
	4. Desde el ciente 2 buscar en la lista de jugadores al cliente 1 y mandarle una invitacion.
	5. Aceptar la invitacion desde el cliente 1.

Todo esto se hace desde distintas terminales (una por cliente y servidor).
